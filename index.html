Websql

Dynamic generation of web site from sql queries

View project onGitHub

Create your site in few minutes

WebSql allows you to create a dynamic Web Site by entering SQL Queries. To create a new page that lists all clients :

ListPage("clients") fromQuery "Select * from Client"

Generate html, xml, json or csv

Html pages are rendered either based on standard templates or by creating a specific template. Specific templates are easy to create with an intuitive syntax.

Xml, json or csv data structure can be generated by explicitly specifying the format.

http://www.yoursite.com/clients
http://www.yoursite.com/clients?format=xml
http://www.yoursite.com/clients?format=json
http://www.yoursite.com/clients?format=csv

Live edit your site in the browser

Change your site without any editor directly in the browser by clicking the edit site button.

Need some user specific content ?

Just provide an authentication query for your Web Site :

authentication="select login from users 
 where login = {login} and password = {password}"

And secure pages as you want !

ListPage("accounts") 
 fromQuery "Select * from accounts where user = {username}" 
 withAuthentication
