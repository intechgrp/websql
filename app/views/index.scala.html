Websql


Dynamic Web Site generation based on SQL Queries

Create your site in few minutes

WebSql allows you to create a dynamic Web Site by entering SQL Queries.
To create a new page that lists all clients : 

ListPage("clients") fromQuery "Select * from Client"

Generate html, xml or json

You can generate html rendering either based on standard templates or by creating a specific template. Specific templates are easy to create with our intuitive syntax.
You can also generate xml or json rendering by explicitly specifying the format. 

http://www.yoursite.com/client/1
http://www.yoursite.com/client/1?format=xml
http://www.yoursite.com/client/1/?format=json

Need some user specific content?

Just provide an authentication query for your Web Site : 

authentication="select login from users where login = {login} and password = {password}"

And secure pages as you want ! 

ListPage("accounts") fromQuery "Select * from accounts where user = {username}" withAuthentication
